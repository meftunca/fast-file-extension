### Spesific Escape String Literals

- Object IS : \O /A or \{ /}
- Array IS : \A /A or \[ /]
- String IS : \( /)
- Number IS : \N /N
- Object Key IS : \OK /OK
- BOOLEAN IS : true or false 
  -  \BT /BT or \BF /BF
- NULL IS : \N /N or null
- ...
